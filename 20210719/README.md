### js

sort 함수 원리 a(next) b(prev) a-b 는 b가 더 클때만 -라서 바뀌는데 b는 prev니까 prev가 더 클때만 바뀐다.
즉 오름차순 1,2,3,5...의 결과를 얻는다.
b-a는 a가 클때만 바뀌는데 a는 next니까 next가 더 클때만 바뀐다 즉 5,3,2,1 내림차순이다.
